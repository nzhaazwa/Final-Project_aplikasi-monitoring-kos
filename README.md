# MANUAL PENGGUNA APLIKASI MONITORING OPERASIONAL KOS BERBASIS DESKTOP
**1. PENDAHULUAN**
Manual pengguna ini dibuat untuk membantu pengguna memahami cara menggunakan aplikasi Monitoring Operasional Kos. Panduan ini menjelaskan langkah-langkah dasar untuk menjalankan aplikasi dan menggunakan fitur-fitur yang tersedia, sehingga aplikasi dapat digunakan dengan mudah tanpa memerlukan pengetahuan teknis khusus.


**2. PERSYARATAN SISTEM**
Sebelum menjalankan aplikasi, pastikan perangkat memenuhi persyaratan berikut:
1) Sistem Operasi: Windows
2) Perangkat telah terinstal aplikasi KOSMON-Setup (.exe)
3) Koneksi internet aktif (untuk akses database)


**3. CARA MENJALANKAN APLIKASI**
1) Klik dua kali ikon aplikasi KOSMON pada desktop atau menu Start.
2) Tunggu hingga aplikasi terbuka dan menampilkan halaman utama.
3) Setelah aplikasi terbuka, pengguna dapat langsung menggunakan fitur-fitur yang 
tersedia.


**4. PENGENALAN ANTARMUKA APLIKASI**
Aplikasi terdiri dari tiga menu utama berbentuk tab, yaitu:
1) Tab Kamar
2) Tab Penghuni
3) Tab Pembayaran
Setiap tab memiliki form input, tombol aksi, dan tabel data untuk memudahkan pengelolaan informasi


**5. PANDUAN PENGGUNAAN FITUR
  5.1 Mengelola Data Kamar**
  Digunakan untuk mengelola informasi kamar kos.
  Langkah-langkah:
  1) Pilih tab Kamar.
  2) Isi Nomor Kamar dan Lantai.
  3) Pilih Status Kamar (Kosong / Terisi).
  4) Klik tombol Tambah untuk menyimpan data kamar.
  5) Untuk memperbarui status, pilih data pada tabel lalu klik Perbarui Status.
  6) Untuk menghapus data kamar, pilih data lalu klik Hapus.
  7) Gunakan fitur filter untuk menampilkan kamar Kosong atau Terisi.
     
  **5.2 Mengelola Data Penghuni**
  Digunakan untuk mencatat data penghuni kos.
  Langkah-langkah:
  1) Pilih tab Penghuni.
  2) Isi Nama Penghuni, Nomor Kamar, Tanggal Masuk, dan Kontak.
  3) Klik tombol Tambah untuk menyimpan data.
  4) Sistem akan otomatis mengubah status kamar menjadi Terisi.
  5) Untuk menghapus data penghuni, pilih data lalu klik Hapus.
  6) Sistem akan otomatis mengubah status kamar kembali menjadi Kosong.
  7) Gunakan kolom pencarian untuk mencari data penghuni berdasarkan nama dan 
  nomor kamar.
  
  **5.3 Mengelola Data Pembayaran**
  Digunakan untuk mencatat dan memantau pembayaran kos.
  Langkah-langkah:
  1) Pilih tab Pembayaran.
  2) Isi Nama Penghuni, Tanggal Jatuh Tempo, Status Pembayaran, dan Keterangan.
  3) Klik tombol Tambah untuk menyimpan data pembayaran.
  4) Untuk memperbarui status pembayaran, pilih data lalu klik Perbarui Status.
  5) Untuk menghapus data pembaran, pilih data lalu klik Hapus.
  6) Gunakan filter untuk menampilkan pembayaran Lunas atau Belum Lunas.


**6. DIALOG BOX DAN NOTIFIKASI**
Aplikasi menggunakan dialog box untuk memberikan interaksi dan peringatan kepada 
pengguna, antara lain:
1) Dialog peringatan jika input belum lengkap
2) Dialog konfirmasi sebelum menghapus data
3) Dialog informasi saat data berhasil disimpan
4) Dialog reminder otomatis jika terdapat pembayaran yang terlambat lebih dari 3 hari

   
**7. PENUTUP**
Dengan mengikuti panduan pada manual pengguna ini, diharapkan pengguna dapat 
mengoperasikan aplikasi Monitoring Operasional Kos dengan baik dan memanfaatkan 
seluruh fitur yang tersedia untuk membantu pengelolaan kos secara lebih rap
